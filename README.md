# Titanic Survival Prediction

## About
This is a beginner machine learning project where I tried to predict whether a passenger survived the Titanic disaster.

## Dataset
The dataset includes details like:
- Age
- Gender
- Passenger class
- Fare
- Survival status

## What I did
- Loaded the dataset
- Cleaned missing values
- Converted text data into numbers
- Split data into training and testing
- Trained a Logistic Regression model
- Checked accuracy

## Result
The model gives around 75% accuracy.

## Tools Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

This project was done as part of CodSoft Internship Task 1.
